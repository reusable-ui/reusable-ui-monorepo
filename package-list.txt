Default inheritance case example:
```tsx
<Group orientation='inline' size='lg' theme='primary' gradient outlined mild>
    {/* default props: size='inherit' theme='inherit' gradient='inherit' outlined='inherit' mild='inherit' */}
    <Label>Quantity:</Label>
    <Button>➖</Button>
    <Input />
    <Button>➕</Button>
    
    <Label>Stock level:</Label>
    
    {/* default prop for orientation is static to inline or block, depends on component's default, except if explicity set to inherit */}
    <Progress>
        <ProgressBar value={70}>Ready</ProgressBar>
        <ProgressBar value={5}>Broken</ProgressBar>
    </Progress>
</Group>
```

|------------------|----------------------------------------------------|---------------------------------------------------------------------|
| Package Name     | Recommended Behavior when Disabled                 | Reason                                                              |
|------------------|----------------------------------------------------|---------------------------------------------------------------------|
| focus-state      | Reset lost; remain lost until user refocuses       | Disabled elements shouldn’t retain focus; accessibility consistency |
| hover-state      | Reset lost; re-evaluate on re-enable via `match()` | Matches native browser hover behavior under cursor                  |
| press-state      | Reset lost; remain lost until user re-presses      | Prevents unintended actions; press must be explicit                 |
| validity-state   | Unvalidated; recompute on re-enable                | Disabled inputs excluded from validation; recompute when re-enabled |
| collapse-state   | Block dispatch; preserve last state                | Disabled freezes interaction but doesn’t reset                      |
| active-state     | Block dispatch; preserve last state                | Disabled freezes interaction but doesn’t reset                      |
| view-state       | Block dispatch; preserve last state                | Disabled freezes interaction but doesn’t reset                      |
| excite-state     | Unaffected                                         | Purely notification/animation                                       |
|------------------|----------------------------------------------------|---------------------------------------------------------------------|

|------------------|----------------------------------------------------|---------------------------------------------------------------------|
| Package Name     | Recommended Behavior when Readonly                 | Reason                                                              |
|------------------|----------------------------------------------------|---------------------------------------------------------------------|
| focus-state      | Unaffected; focus feedback applies                 | Readonly inputs are still part of tab order and accessible          |
| hover-state      | Unaffected; hover feedback applies                 | Pointer feedback remains useful for discoverability                 |
| press-state      | Unaffected; press feedback applies                 | Press can select/focus but must not trigger state changes           |
| validity-state   | Unvalidated; recompute on re-enable                | Readonly inputs excluded from validation; recompute when re-mutable |
| collapse-state   | Block dispatch; preserve last state                | Readonly prevents mutation; interaction state frozen                |
| active-state     | Block dispatch; preserve last state                | Same rationale as collapse                                          |
| view-state       | Block dispatch; preserve last state                | Same rationale as collapse                                          |
| excite-state     | Unaffected                                         | Purely notification/animation                                       |
|------------------|----------------------------------------------------|---------------------------------------------------------------------|
